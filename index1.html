My GIT Cheatsheet

The website describes the basic GIT commands that is required to manage the code

All the GIT commands are executed in the command line.

The below information described how to use the GIT command.

GIT - Clone

The command is used to obtain a repository from an existing URL

Example: https://github.com/adenijiahmed/git-cheatsheet.git

GIT - Status

The command lists all the files that have to be committed.

GIT - Add

The command adds a file to the staging area.

Example: Desktop/lesson12/git-cheatsheet/ git add Index.html

GIT - Commit

The command records or snapshots the file permanently in the version history.

Example: Desktop/lesson12/git-cheatsheet/ git commit -m "Index.html"

GIT - Push

The command sends the committed changes of master branch to the remote repository.

Example: Desktop/lesson12/git-cheatsheet/ git puch origin master Index.html
